# 顺序表的应用
## 一、集合的“并”运算
将B中不存在于A的元素插入A尾部，最终A保存两集合的并集。
```cpp
void Union(SeqList &A, SeqList B) {
    int n = ListLength(A);
    int m = ListLength(B);
    for(int i = 1; i <= m; i++){
        ListData x = GetElem(B, i);// 在B中取一元素,用的动态分配的GetElem
        int k = LocateElem(A, x);// 在A中查找它
        if (k == -1) {// 若未找到，插入它到尾部
            ListInsert(A, n + 1, x);
            n++;
        }
    }
}
```

## 二、集合的“交”运算
删除A中不存在于B的元素，最终A保存两集合的交集。
```cpp
void Intersection(SeqList &A, SeqList B) {
    int n = ListLength(A);// 在A中取第i个元素
    int m = ListLength(B);// 在B中查找
    int i = 1;
    while (i <= n){
        ListData x = GetElem(A, i);
        int k = LocateElem(B, x);
        if (k == -1) {
            ListDelete(A, i);
            n--;
        }
        else i++;
        // 删除会覆盖前方元素，故Delete后i不变
    }
}
```
